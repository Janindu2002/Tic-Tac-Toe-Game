# Algorithm

1. Initialize a 3×3 board with numbers 1–9.
2. Set the first player as Player 1 (symbol X).
3. Repeat until game ends:
   - Display the board.
   - Ask the current player to choose a position.
   - Validate the position.
   - Place the player's symbol in that position.
   - Check if the player has won.
   - If won:
       - Display board.
       - Announce winner.
       - End game.
   - Check if board is full.
   - If full:
       - Display draw message.
       - End game.
   - Switch player.
4. End program.
