# Edge Cases

## 1 Invalid Position
Player enters number outside 1–9.

Handling:
Prompt user again.

---

## 2 Cell Already Occupied
Player selects a position already taken.

Handling:
Display error and request new input.

---

## 3 Early Win
Game stops immediately once a player wins.

---

## 4 Draw Condition
If all 9 cells filled without winner → draw.

---

## 5 Non-numeric Input
User inputs invalid characters.

Handling:
Clear input buffer and request valid input.
