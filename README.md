# Tic Tac Toe Game (C++)

## Overview
This project implements a simple command-line Tic Tac Toe game using C++. Two players take turns placing their marks (X and O) on a 3×3 grid. The goal is to align three marks in a row, column, or diagonal.

This project demonstrates:
- Basic C++ programming
- Algorithm design
- Game logic implementation
- Structured documentation for software projects

## Features
- Two-player gameplay
- Input validation
- Win detection
- Draw detection
- Clean board display

## Technologies
- C++
- Standard Library (iostream)
