# Problem Definition

## Problem
Create a console-based Tic Tac Toe game in C++ where two players can play by taking turns.

## Inputs
- Player position input (1–9)

## Outputs
- Game board after every move
- Winner announcement
- Draw message if board fills without winner

## Constraints
- Board size fixed to 3×3
- Players cannot overwrite an occupied cell
- Game stops immediately when a player wins

## Goal
Implement a simple interactive game while demonstrating structured algorithmic design and documentation.
